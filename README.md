# probot-hello-app

> A GitHub App built with [Probot](https://github.com/probot/probot) that A Probot app.
> Used for testing changes to Probot framework.

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
